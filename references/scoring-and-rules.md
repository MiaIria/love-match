# Scoring And Insight Rules

## Goal

The scoring layer turns answers into explainable emotional patterns. It should
support insight generation without pretending to be a clinical diagnosis.

## Core Dimensions

The assessment can be interpreted through seven relationship dimensions:

| Dimension | Meaning |
| --- | --- |
| Social energy | How the user gains or spends energy around others. |
| Emotional expression | How directly the user reveals feelings. |
| Security need | How much stability and reassurance the user needs. |
| Conflict style | How the user responds to tension or disagreement. |
| Independence balance | How the user balances closeness and autonomy. |
| Value orientation | What the user prioritizes in life and relationships. |
| Connection depth | Whether the user prefers light companionship or deep bonding. |

These dimensions are tendencies, not identity labels.

## Answer Mapping

Each answer should affect one or more dimensions.

Example:

```text
When conflict appears:
A. Ask directly what happened
   -> higher emotional expression, higher conflict directness
B. Observe first and wait for more signals
   -> medium security need, lower directness
C. Hide the emotion but keep thinking about it
   -> higher security need, lower expression
D. Pull back quickly to protect yourself
   -> higher independence, lower openness
```

## Insight Rules

Rules detect cross-dimensional patterns. They prevent the report from becoming a
loose collection of generic statements.

Rule categories:

| Category | Use |
| --- | --- |
| Defense and protection | Detect how the user protects vulnerability. |
| Tension and contradiction | Detect mixed patterns, such as wanting closeness while fearing loss of control. |
| Love language | Detect what kind of care the user recognizes. |
| Security pattern | Detect what calms or alarms the user. |
| Connection depth | Detect the preferred depth and rhythm of bonding. |

## Example Rule

Trigger:

- low direct emotional expression,
- high security need,
- withdrawal under conflict.

Insight:

```text
You may not be cold. You may need a longer runway before trusting someone with
your softer emotions.
```

## Quality Rule

Every important insight should be traceable to:

- a specific answer,
- a cross-question pattern,
- or a defined rule.

If it cannot be traced, it should be removed or softened.

## Interview Talking Point

The point of the scoring layer is not to quantify love precisely. It is to make
the analysis explainable enough that the user can see why the report reached a
given conclusion.
